## Giới thiệu

Dự án MediWOW được xây dựng nhằm mục đích hỗ trợ công tác quản lý bán hàng tại tiệm thuốc tây bản lẻ, giúp tối ưu hóa quy trình làm việc và nâng cao hiệu quả quản lý.

## Chức năng chính

- Quản lý thông tin sản phẩm.
- Quản lý thông tin khuyến mãi.
- Quản lý thông tin khách hàng thành viên.
- Quản lý tài khoản dược sĩ.
- Quản lý bán hàng và thống kê báo cáo.

## Yêu cầu hệ thống

- Hệ điều hành: Windows
- Java 21 (JDK21)
- Cơ sở dữ liệu: Microsoft SQL Server

## Cài đặt & chạy dự án

1. Clone repository:
    ```bash
    git clone https://github.com/hauuto/MediWOW-Pharmacy.git
    ```
2. Mở IntelliJ IDEA, chọn **Open** và trỏ tới thư mục dự án vừa clone.
3. Đảm bảo IntelliJ đã nhận diện dự án là Maven Project (có file `pom.xml`).
4. Chờ Maven tải dependencies (thư viện) tự động.
5. Cấu hình kết nối cơ sở dữ liệu SQL Server trong file cấu hình (nếu có).
6. Nhấn chuột phải vào file `Main.java` và chọn **Run** để chạy chương trình.
